# 🌱 NayePankh Foundation Website Redesign

A modern, responsive redesign of the NayePankh Foundation website built with **React** and **Tailwind CSS**. The project focuses on improving the user experience, accessibility, responsiveness, and overall visual design while preserving the organization's mission and identity.

## 📖 Overview

This project was created as part of the **NayePankh Foundation Full Stack Development Internship Task**. The goal was to redesign the existing website with a clean, modern interface and propose improvements that enhance usability and engagement.

## ✨ Features

- 🎨 Modern and responsive UI
- 🌙 Dark Mode support
- 📱 Mobile-first design
- ⚡ Smooth animations
- 📊 Impact statistics section
- ❤️ Volunteer and Donation call-to-actions
- 📰 Recognition & Certificates section
- 💬 Testimonials section
- 📬 Newsletter subscription
- 📞 Contact section
- 🦶 Improved footer with better spacing and navigation

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- JavaScript (ES6+)
- Vite
- Lucide React Icons
- Framer Motion (Animation)

## 🚀 Improvements Over the Existing Website

- Reduced header height for better content visibility.
- Improved spacing throughout the website.
- Added proper spacing before the footer.
- Redesigned navigation for desktop and mobile devices.
- Enhanced typography and visual hierarchy.
- Modern card layouts and section organization.
- Improved accessibility and responsiveness.
- Better call-to-action buttons for volunteers and donors.

## 🏃 Getting Started

Clone the repository:

```bash
git clone https://github.com/jawad-js/nayepankh-foundation-redesign.git
```

Navigate to the project:

```bash
cd nayepankh-redesign
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
